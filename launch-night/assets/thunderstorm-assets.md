## 3. CONTENT ID SAFETY PROTOCOL (run before the 2am public launch)

1. **Cut a 30-second test clip** combining the candidate video + the candidate audio (use the actual encode settings you'll stream with — same codec, bitrate, sample rate).
2. **Upload as UNLISTED** to YouTube Studio (`youtube.com/upload`), set monetization ON, leave it for ~10–15 min for Content ID to process.
3. **Check the "Checks" / "Copyright" tab** on the video. Status must be "No issues found". Any "Copyrighted content detected" → discard that asset and try the next, **even if the claim is "released for monetization"** — claims still split revenue.
4. **Repeat for every video/audio pair** before locking the 24/7 stream input.
5. **After launch**, keep an unlisted "canary" re-upload of the same clip running so you get notified if a new Content ID match is added to the DB days later.

Optional belt-and-suspenders: run each audio asset through https://www.tunebat.com/ or upload to a private SoundCloud account to detect Audible Magic matches independently of YouTube.
